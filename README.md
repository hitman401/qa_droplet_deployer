# qa_droplet_deployer

## Prerequisite

[Nodejs](https://nodejs.org/en/download/) should be installed.
 
## Setting up instruction

Clone the repository and run 'npm install'.

## Usage

'npm start' - Follow the default flow of the tool as listed [here](/script_flow.md)
